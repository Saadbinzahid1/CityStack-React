# 🌍 CityStack – Travel Tracker with Interactive Map

CityStack is a React single-page application that allows users to track the cities they have visited and visualize them on an interactive map.

The project focuses on building a dynamic and responsive travel tracking experience while practicing modern React concepts.

## ✨ Features

- 📍 Add cities you have visited
- 🗺️ Visualize cities on a real-time map
- 📋 Maintain a list of visited locations
- ⚡ Dynamic UI updates
- 🔄 Smooth navigation with React SPA architecture
- 📱 Clean and responsive interface

---

## 🚀 Technologies Used

- React (Functional Components)
- React Router
- JavaScript (ES6+)
- CSS3
- Interactive Map Library (e.g. Leaflet / Mapbox)
